# Hero-Squad App

## Description
This is an app whereby A user Creates a Hero and Assigns the hero t a Squad.

## prerequisites

1. Uses Jdk inorder for it to be compiled.
2. Uses Gradle inorder to built.
3.USes heroku to be deployed on Github.
